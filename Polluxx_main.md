# 🌟 OMNY-Twin (Polluxx) 模型架构核心摘要
*一页纸 A4 极简报告：医疗大模型从理论到落地的工程结晶*

---

## 🎯 核心定位与规模
- **定位**：专为**电子健康记录 (EHR)** 设计的纵向时序预测基础大模型。
- **目标**：精准学习患者数年的健康轨迹，预测未来临床事件与疾病进展。
- **规模**：**1.38B** (总参数) / **322M** (前向活跃) 👉 *以 23.3% 激活率实现 4.3× 参数效率*。
- **训练数据**：8700 万条预分词医疗记录 (最大并发长度 4096)。

---

## 🏗️ 核心架构：[ Mamba × Attention × MoE ] 混合引擎
创新性采用 **混合型 Decoder-only自回归架构**，层级模式为 **[M, M, M, A] × 4** (共 16 层)。

### 1️⃣ Mamba State Space (主干，占比 75%)
- **优势**：线性时间复杂度 $O(n)$，内存节省超 50%。
- **职责**：宏观捕获患者的**长期时序依赖**（跨月/年的纵向健康发展）。

### 2️⃣ Attention 机制 (点睛，融合于全局)
- **优势**：极低的 Attention 频率 ($O(n^2)$ 计算量被压缩到 25% 层数)。
- **职责**：微观精算**单次就诊 (Intra-visit)** 的复杂临床特征交互。

### 3️⃣ Sparse MoE (专家路由，医疗专业化)
- **配置**：**8 专家系统**，采用 **Top-2** 动态路由 (每次前向只激活 25%)。
- **职责**：针对不同病患特征（心血管 / 神经 / 肿瘤等）自动激活对应医疗专家网络，有效提升 4 倍知识容量而**不增加计算负担**。

## 🗺️ 极简数据流与拓扑结构

### 1. 数据管线 (Tokenization & Binning)
```text
[ 真实原始 EHR 医疗记录 ] 
           │
           ▼
⚙️ 规范化映射 (Binning & Encoding)
 ├── 静态特征 (年龄/性别等) ───▶ [ Static Tokens (5 个) ]
 ├── 连续时间特征 (天数差) ───▶ [ ContinuousTimeEncoder (生成与主干等维的 1280 维向量) ]
 └── 诊断/体征/检验/药物 ────▶ [ Clinical Tokens (严格排序) ]
           │
           ▼
📦 序列装箱与特征融合 (Sequence Packing & Time Fusion)
 [ 患者A: Static + <start> + [Enc]* + <end> ] + [ 患者B: ... ]
 (注：提取的时间特征会直接与就诊记录的 Token Embedding 相加融合，不再占用独立序列位置)
```

                          │
  ┌───────────────────────┼───────────────────────┐ 
  │ 🔄 循环 4 次          ▼                       │
  │     ╔═══════════════════════════════════╗     │
  │     ║ 🟢 Mamba Block 1 (线性时间复杂度) ║     │
  │     ║ 🟢 Mamba Block 2 (状态空间记忆)   ║     │ 
  │     ║ 🟢 Mamba Block 3 (捕捉纵向长依赖) ║     │
  │     ╚═════════════════╤═════════════════╝     │
  │                       ▼                       │
  │     ╔═══════════════════════════════════╗     │

## ⚡ 性能与硬件极限优化
- **张量对齐 (Tensor Core)**：词表极限填充至 `26,880` (64/128倍数)，B200 GPU 性能暴增 10-15%。
- **极致内存管理**：引入 Gradient Checkpointing (梯度检查点), 显存占用骤降 73% (约 12GB/卡)。
- **稳健数值传播**：全面启用 **RMSNorm (Pre-Norm)** 与 **bfloat16** 加持，省去均值计算同时保障不崩盘。
- **参数共享**：LM Head 与 Embedding 权重绑定，直接“白嫖” 34M 专业医学词库空间记忆。
